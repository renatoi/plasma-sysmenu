# Changelog

All notable changes to this project will be documented in this file.
The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and versions follow the `Version` field in `metadata.json` (tag `v<version>`).

## [1.1] - 2026-07-15

### Added
- "Restore Defaults" action in the settings page's title bar.

## [1.0] - 2026-07-15

### Added
- Initial release: macOS-style Apple menu for the Plasma panel.
- Dropdown with About This Computer, System Settings, Discover, Force Quit
  (KWin interactive kill), Sleep, Restart, Shut Down, Lock Screen and Log Out.
- Configurable panel icon via KDE's icon dialog (theme icons or custom
  PNG/SVG files) and icon size (fit panel or fixed).
